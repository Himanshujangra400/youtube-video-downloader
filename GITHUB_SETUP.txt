GITHUB SETUP INSTRUCTIONS
=========================

Follow these steps to push your project to GitHub:

STEP 1: Create GitHub Repository
================================
1. Go to https://github.com/new
2. Repository name: youtube-clip-downloader
3. Description: Fast Electron desktop app for downloading YouTube clips with time-range extraction and batch processing
4. Choose "Public" (so everyone can see it)
5. DO NOT check "Initialize with README" (we already have one)
6. Click "Create repository"
7. Copy the HTTPS URL from the new page (looks like: https://github.com/YOUR_USERNAME/youtube-clip-downloader.git)

STEP 2: Initialize Git Locally
==============================
Open PowerShell in the project folder:
  cd "c:\Users\ZXproNoob\Desktop\youtube video downloader"

Then run:
  git config --global user.name "Your Name"
  git config --global user.email "your.email@example.com"
  git init
  git add .
  git commit -m "Initial commit: YouTube Clip Downloader - Fast batch downloading with time-range clipping"

STEP 3: Push to GitHub
======================
Run these commands (replace YOUR_USERNAME with your actual GitHub username):

  git branch -M main
  git remote add origin https://github.com/YOUR_USERNAME/youtube-clip-downloader.git
  git push -u origin main

STEP 4: Verify It's Public
==========================
1. Go to https://github.com/YOUR_USERNAME/youtube-clip-downloader
2. Check that it says "Public" (not Private)
3. If it says "Private", click Settings → scroll down → change to "Public" → Save

DONE! ✅
========
Your project is now public on GitHub. Anyone can clone it with:
  git clone https://github.com/YOUR_USERNAME/youtube-clip-downloader.git
